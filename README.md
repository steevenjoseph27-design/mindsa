MindGuard 🧠
Application de bien-être mental avec détection précoce du burnout et de la dépression.
Structure du projet
```
mindguard/
├── index.html          ← Page principale
├── package.json        ← Dépendances
├── vite.config.js      ← Configuration Vite
└── src/
    ├── main.jsx        ← Point d'entrée React
    ├── App.jsx         ← Routeur principal
    ├── Auth.jsx        ← Inscription / Connexion (avec RGPD)
    └── Dashboard.jsx   ← Application principale
```
Installation locale
```bash
npm install
npm run dev
```
Ouvre http://localhost:5173
Déploiement sur Vercel
Crée un repo GitHub et pousse ce dossier
Va sur vercel.com → Add New Project
Importe ton repo GitHub
Framework : Vite (détecté automatiquement)
Clique Deploy ✅
